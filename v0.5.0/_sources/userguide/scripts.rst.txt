Executables and Scripts
=======================

``saltproc``
------------
Once you have a model built (see :ref:`userguide_basics`), you can run SaltProc
from the command line!

.. code-block:: bash

   python -m saltproc -i saltproc_input.json


``saltproc`` accepts the following command line flags:

-s, --threads   Threads to use for shared-memory parallelism
-h, --help      Show help message

Other scripts
-------------
Additional scripts can be found in the :ref:`scripts directroy <https://github.com/arfc/saltproc/tree/master/scripts>.
