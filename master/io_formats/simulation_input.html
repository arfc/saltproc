simulation Properties¶

description

Name of simulation

type

string

db_name¶

description

Output HDF5 database file name

type

string

default

saltproc_results.h5

pattern

^(.*)\\.h5$

restart_flag¶

description

Restart simulation from the step when it stopped?

type

boolean

default

false

description

switch to another geometry when keff drops below 1?

type

boolean

default

false
